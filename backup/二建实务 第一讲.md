# 第 1 篇 市政公用工程技术

## 第 1 章 城镇道路工程

### 1.1 道路结构特征

1. 城镇道路分类

      1. 城镇道路在道路网的地位、交通功能、对沿线的服务功能划分为<font color='red'>快速路、主干路、次干路及支路</font> 。

         

         <font color='red'>快速路又称城市快速路，完全为交通功能服务，是解决城市大容量、长距离、快速交通的主要道路。</font>

         

         <font color='red'>主干路以交通功能为主，为连接城市各主要分区的干路，是城市道路网的主要骨架。 </font>

         

         <font color='red'>次干路是城市区域性的交通干道，为区域交通集散服务，兼有服务功能，结合主干路组成干路网。</font>

         

         <font color='red'>支路为次干路与居住小区、工业区、交通设施等内部道路的连接线路，解决局部地区交通，以服务功能为主。</font>

      2. 根据路面材料划分 可分为<font color='red'>沥青路面、水泥混凝土路面和砌块路面。</font>

         | 路面           | 分类                                                  | 适用范围                            |
         | -------------- | ----------------------------------------------------- | ----------------------------------- |
         | 沥青路面       | 沥青混凝土                                            | 各交通等级道路                      |
         |                | 沥青贯入式、沥青表面处治                              | 支路、停车场                        |
         | 水泥混凝土路面 | 普通混凝土、钢筋混凝土、连续配 筋混凝土、钢纤维混凝土 | 各交通等级道路                      |
         | 砌块路面       | 石材、预制混凝土砌块                                  | 支路、广场、停车场、人行道与步 行街 |

      3. 按路面力学性能划分 

         可分为刚性和柔性路面。

         刚性路面主要代表有：水泥混凝土路面；

         柔性路面主要代表有：各类沥青类路面。

     ### 1.1.1 道路路基结构特征 

      1.路基分类 
      
      根据材料不同，路基可分为<font color='red'>土方路基、石方路基、特殊土路基</font>。
      
      路基断面形式分类有：
      
      路堤--路基顶面高于原地面的填方路基；
      
      路堑--全部由地面开挖出的路基； 
      
      半填半挖路基--横断面一侧为挖方，另一侧为填方的路基。 
      
      补充解释：路基按干湿状态不同，分为干燥、中湿、潮湿和过湿四类。
      
      2.路基填料的要求
      
      高液限黏土、高液限粉土及含有机质的细粒土，不适于做路基填料。因条件限制而必须采用上述土做填料时，应	掺加石灰或水泥等结合料进行改善。地下水位高时，宜提高路基顶面标高。在设计标高受限制，未能达到中湿状	态的路基临界高度时，应选用粗 粒土或低剂量石灰或水泥稳定细粒土做路基填料，同时应采取在边沟下设置排水	渗沟等降低地下水位的措施。
      
      3.路基的功能和性能要求 
      
      （1）路基既为车辆在道路上行驶提供基础条件，也是道路的支撑结构物，对路面的使用性能有重要影响。路 基	应稳定、密实、均质，对路面结构提供均匀的支承，即路基在环境和荷载作用下不产生不均匀变形。 
      
      （2）性能主要指标：路基整体稳定性和变形量
